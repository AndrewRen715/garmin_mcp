# Garmin 跑步分析技能

此技能帮助分析 Garmin 跑步活动并提供训练洞察。

## 功能

- 分析来自 Garmin Connect 的最近跑步活动，包括每公里分段数据
- 使用 me.md 中的个人心率区间进行准确的强度分析
- 生成全面的训练状态报告和表现趋势
- 根据分析提供个性化训练建议
- 提供详细的表现洞察和改进建议

## 何时调用

当以下情况时调用此技能：
- 用户要求分析最近的跑步活动
- 用户想检查训练状态或进度
- 用户请求关于跑步表现的洞察
- 用户需要训练改进建议

## 使用示例

### 示例 1：分析最近跑步

**用户查询：** "分析我最近的跑步活动"

**技能动作：**
- 运行 `analyze_recent_run_with_splits.py` 获取最新活动数据、每公里分段和心率分析
- 读取 `me.md` 获取个人心率区间和表现基准
- 结合数据提供全面分析
- 生成训练状态报告和表现趋势

### 示例 2：分析昨天的跑步

**用户查询：** "查看一下我昨天的跑步并给出分析"

**技能动作：**
- 运行 `analyze_yesterday_training.py` 获取昨天的训练数据
- 分析心率数据、训练负荷和恢复指标
- 提供详细的活动分析和健康指标
- 生成个性化的训练建议

### 示例 3：检查训练状态

**用户查询：** "我的训练状态如何？"

**技能动作：**
- 运行 `get_training_status.py` 获取当前训练状态
- 分析最近的训练负荷和恢复情况
- 提供状态摘要和建议

## 实现细节

此技能利用以下脚本和文件：
- `analyze_recent_run_with_splits.py` - 获取最新活动数据、每公里分段和心率分析
- `analyze_yesterday_training.py` - 分析昨天的训练数据
- `get_training_status.py` - 获取当前训练状态数据
- `analyze_training_data.py` - 分析训练趋势和模式
- `me.md` - 包含个人心率区间和表现基准

## 输出格式

调用此技能时，提供以下格式的输出：

1. **活动摘要：** 分析跑步的关键指标，包括每公里分段
2. **表现分析：** 基于 me.md 中的个人心率区间的详细分析
3. **训练状态报告：** 当前训练状态和负荷
4. **表现趋势：** 最近表现模式的分析
5. **训练建议：** 基于分析的个性化建议
6. **后续步骤：** 潜在的后续行动

## 语言支持

此技能支持基于用户偏好的中英文输入/输出。
