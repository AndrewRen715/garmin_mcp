## 项目概述

本系统旨在帮助跑者：

- 识别和预防常见跑步伤病
- 科学管理伤病康复过程
- 基于伤病记录智能调整训练计划
- 安全回归正常训练

通过定期查询最近1-2个月的伤病记录，系统能够为跑者提供个性化的训练建议，确保训练的安全性和有效性。

## 常见跑步伤病

### 下肢伤病

- **髂胫束综合征 (ITBS)**：
  - **症状**：膝盖外侧疼痛，尤其是下坡跑时
  - **原因**：髂胫束过度紧张，与股骨外髁摩擦
  - **风险因素**：跑步姿势不良，肌肉不平衡，突然增加跑量

- **髌股关节疼痛综合征 (PFPS)**：
  - **症状**：膝盖前方疼痛，上下楼梯或蹲下时加重
  - **原因**：髌骨轨迹异常，股四头肌不平衡
  - **风险因素**：过度训练，肌肉力量不足，扁平足

- **胫骨应力综合征 (Shin Splints)**：
  - **症状**：胫骨内侧疼痛，跑步开始时明显
  - **原因**：小腿肌肉过度使用，骨膜炎症
  - **风险因素**：突然增加跑量，硬地面跑步，不合适的鞋子

- **跟腱炎**：
  - **症状**：跟腱部位疼痛、肿胀
  - **原因**：跟腱过度拉伸，炎症
  - **风险因素**：突然增加速度训练，下坡跑过多，跟腱柔韧性差

- **足底筋膜炎**：
  - **症状**：足底疼痛，尤其是早晨起床时
  - **原因**：足底筋膜过度拉伸，炎症
  - **风险因素**：扁平足，高弓足，硬地面跑步

### 上肢与躯干伤病

- **下背部疼痛**：
  - **症状**：腰部疼痛，跑步后加重
  - **原因**：核心力量不足，跑步姿势不良
  - **风险因素**：长距离跑，核心肌肉薄弱

- **肩关节不适**：
  - **症状**：肩部酸痛，摆臂时加重
  - **原因**：摆臂姿势不良，肌肉紧张
  - **风险因素**：过度训练，肩部柔韧性差

## 伤病预防

### 训练管理

- **渐进超负荷**：每周跑量增加不超过10-15%，遵循最近4周平均跑量的110%原则
- **合理安排恢复**：确保充足的休息日和轻松跑
- **交叉训练**：定期进行游泳、骑行等低冲击活动
- **强度控制**：避免连续进行高强度训练

### 技术与装备

- **跑步姿势**：保持放松的身体姿态，适当的步幅和步频
- **鞋子选择**：根据脚型和跑步风格选择合适的跑鞋，定期更换（每500-800公里）
- **场地选择**：避免长期在硬地面或倾斜路面跑步
- **热身与放松**：每次训练前充分热身，训练后进行拉伸

### 力量与柔韧性

- **核心力量训练**：平板支撑、仰卧起坐、侧桥等
- **下肢力量训练**：深蹲、箭步蹲、提踵等
- **柔韧性训练**：静态拉伸、瑜伽、泡沫轴放松
- **平衡训练**：单腿站立、平衡板训练

## 伤病识别与早期预警

### 疼痛分级

- **0级**：无疼痛
- **1级**：轻微疼痛，不影响跑步
- **2级**：中度疼痛，跑步时可忍受
- **3级**：严重疼痛，影响跑步表现
- **4级**：剧烈疼痛，无法跑步

### 早期预警信号

- **持续疼痛**：训练后疼痛不消失，或休息后加重
- **肿胀**：受伤部位出现肿胀
- **活动受限**：关节活动范围减小
- **跑步形式改变**：为避免疼痛而改变跑步姿势
- **力量下降**：受伤部位力量减弱

### 自我监测工具

- **训练日志**：记录训练强度、疼痛程度和位置
- **心率数据分析**：异常心率变化可能提示过度训练
- **配速变化**：突然的配速下降可能与伤病有关
- **步频与步幅分析**：异常变化可能提示跑步形式改变

## 伤病康复训练

### 急性期处理 (0-7天)

- **RICE原则**：休息(Rest)、冰敷(Ice)、加压(Compression)、抬高(Elevation)
- **减轻负荷**：暂停跑步，进行低冲击交叉训练
- **疼痛管理**：适当使用非甾体抗炎药
- **轻柔活动**：保持关节活动范围，避免僵硬

### 亚急性期处理 (7-21天)

- **渐进性力量训练**：针对受伤部位的肌肉进行等长收缩训练
- **柔韧性训练**：轻度拉伸，避免过度拉伸
- **本体感觉训练**：平衡训练，提高神经肌肉控制
- **低强度有氧**：游泳、骑行等，保持心肺功能

### 恢复期处理 (21天以上)

- **功能性训练**：模拟跑步动作的训练
- **专项力量训练**：针对跑步相关肌肉群
- **渐进性负荷**：逐渐增加训练强度和时间
- **跑步特异性训练**：跑步机快走、慢跑等

## 回归训练计划

### 第一阶段：基础恢复 (1-2周)

- **活动**：快走、游泳、骑行
- **频率**：3-4次/周
- **强度**：低强度，无疼痛
- **时间**：20-30分钟/次

### 第二阶段：跑步重新引入 (2-3周)

- **活动**：间歇慢跑（走跑结合）
- **频率**：3次/周
- **强度**：轻松配速，无疼痛
- **时间**：10-15分钟/次，逐渐增加

### 第三阶段：跑步重建 (3-4周)

- **活动**：连续慢跑
- **频率**：3-4次/周
- **强度**：轻松配速，逐渐增加
- **时间**：20-30分钟/次

### 第四阶段：训练强度恢复 (4-6周)

- **活动**：加入速度训练和长距离跑
- **频率**：4-5次/周
- **强度**：逐渐恢复到伤前水平
- **时间**：根据训练计划调整

### 回归注意事项

- **疼痛监测**：任何疼痛超过2级应立即停止
- **渐进原则**：每阶段至少维持1周，无疼痛后再进阶
- **交叉训练**：继续保持交叉训练
- **力量训练**：持续进行针对性力量训练
- **耐心**：完全恢复可能需要3-6个月，避免急于求成

### 最近1-2个月伤病查询

#### 查询方法

1. **手动记录查询**：
   - 查看最近60天的训练日志和伤病记录
   - 整理伤病发生时间、类型、严重程度和恢复情况
   - 评估当前伤病状态和影响

2. **数据文件查询**：
   - 检查 `injury_records.json` 文件中的最近记录
   - 使用以下脚本自动生成最近2个月伤病报告：

```python
#!/usr/bin/env python3
"""
查询最近1-2个月伤病记录
"""

import json
import os
from datetime import datetime, timedelta

def get_recent_injuries(days=60):
    """获取最近指定天数的伤病记录"""
    # 计算起始日期
    start_date = datetime.now() - timedelta(days=days)
    
    # 读取伤病记录文件
    injury_file = 'injury_records.json'
    
    if not os.path.exists(injury_file):
        print("未找到伤病记录文件")
        return []
    
    with open(injury_file, 'r', encoding='utf-8') as f:
        records = json.load(f)
    
    # 筛选最近的记录
    recent_records = []
    for record in records:
        record_date = datetime.fromisoformat(record.get('date'))
        if record_date >= start_date:
            recent_records.append(record)
    
    return recent_records

def generate_injury_report():
    """生成伤病报告"""
    print("=== 最近2个月伤病记录报告 ===")
    
    # 获取最近60天的伤病记录
    recent_injuries = get_recent_injuries(60)
    
    if not recent_injuries:
        print("最近2个月无伤病记录")
        return
    
    print(f"共找到 {len(recent_injuries)} 条伤病记录")
    print()
    
    for i, injury in enumerate(recent_injuries, 1):
        print(f"{i}. 日期: {injury.get('date')}")
        print(f"   伤病: {injury.get('name')}")
        print(f"   部位: {injury.get('location')}")
        print(f"   等级: {injury.get('pain_level')}")
        print(f"   状态: {injury.get('status', '未知')}")
        print()
    
    # 生成训练建议
    print("=== 基于伤病记录的训练建议 ===")
    if recent_injuries:
        # 分析最新伤病
        latest_injury = max(recent_injuries, key=lambda x: x.get('date'))
        pain_level = latest_injury.get('pain_level', 0)
        status = latest_injury.get('status', 'recovery')
        
        if status == 'recovered':
            print("✓ 最近伤病已恢复，可以正常训练")
            print("建议：逐渐恢复训练强度，注意热身和放松")
        elif pain_level <= 1:
            print("⚠ 轻度伤病，建议调整训练")
            print("建议：减少高强度训练，增加交叉训练")
        elif pain_level <= 2:
            print("⚠ 中度伤病，需要调整训练计划")
            print("建议：暂停高强度训练，进行低冲击活动")
        else:
            print("✗ 重度伤病，需要休息和康复")
            print("建议：暂停跑步，进行康复训练")
    
if __name__ == "__main__":
    generate_injury_report()
```

## 训练调整指南

### 轻度伤病调整

#### 调整原则
- **减少强度**：降低训练强度，避免加重伤病
- **保持活动**：维持有氧能力，避免完全停训
- **针对性训练**：增加与伤病相关的康复训练

#### 具体调整
- **跑步**：减少速度训练和长距离跑，以轻松跑为主
- **交叉训练**：增加游泳、骑行等低冲击活动
- **力量训练**：专注于受伤部位的力量和柔韧性训练
- **恢复**：增加休息时间，确保充分恢复

### 中度伤病调整

#### 调整原则
- **暂停高冲击**：暂停跑步等高冲击活动
- **维持体能**：通过交叉训练维持心肺功能
- **积极康复**：进行系统的康复训练

#### 具体调整
- **跑步**：完全暂停或只进行非常轻度的快走
- **交叉训练**：以游泳、骑行、椭圆机为主
- **康复训练**：按照康复计划进行针对性训练
- **频率**：减少训练频率，增加休息时间

### 重度伤病调整

#### 调整原则
- **完全休息**：暂停所有可能加重伤病的活动
- **专业治疗**：寻求专业医疗建议
- **康复为主**：专注于康复训练

#### 具体调整
- **跑步**：完全暂停
- **活动**：只进行医生允许的轻度活动
- **康复**：严格按照医生和物理治疗师的建议进行康复训练
- **监控**：密切关注恢复情况，定期评估

## 案例分析

### 案例1：髂胫束综合征

#### 伤病情况
- **日期**：2026-02-01
- **类型**：髂胫束综合征
- **部位**：膝盖外侧
- **等级**：2级疼痛
- **诱发因素**：突然增加跑量，每周从40km增加到60km

#### 训练调整
- **第一周**：
  - 暂停所有跑步
  - 进行游泳和骑行
  - 进行髂胫束拉伸和臀部力量训练
- **第二周**：
  - 开始短时间快走（10分钟）
  - 继续交叉训练
  - 增加核心力量训练
- **第三周**：
  - 开始间歇慢跑（走30秒，跑30秒，共10分钟）
  - 减少交叉训练时间
  - 继续针对性康复训练
- **第四周**：
  - 增加连续慢跑时间到20分钟
  - 开始轻松跑
  - 逐渐恢复正常训练

#### 结果
- 4周后疼痛完全消失
- 8周后恢复正常训练强度
- 通过调整避免了长期停训

### 案例2：足底筋膜炎

#### 伤病情况
- **日期**：2026-01-15
- **类型**：足底筋膜炎
- **部位**：足底
- **等级**：3级疼痛
- **诱发因素**：穿着旧跑鞋，在硬地面跑步

#### 训练调整
- **第一阶段（1-2周）**：
  - 完全暂停跑步
  - 进行游泳和上肢力量训练
  - 进行足底筋膜拉伸和小腿力量训练
- **第二阶段（3-4周）**：
  - 开始水中跑步
  - 继续非负重训练
  - 增加平衡训练
- **第三阶段（5-6周）**：
  - 开始短距离慢跑（5-10分钟）
  - 减少交叉训练
  - 继续康复训练
- **第四阶段（7-8周）**：
  - 逐渐增加跑量
  - 恢复正常训练结构
  - 持续进行预防训练

#### 结果
- 8周后完全恢复
- 更换了合适的跑鞋
- 通过康复训练增强了足底力量

## 总结
